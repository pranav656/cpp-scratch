# Types of UML diagrams
* Structural Diagrams
  * Class Diagrams: Each class has a bunch of methods and attributes. The visibility is indicated by private(+)/public(-)/protected(#)
  * Component Diagram
  * Deployement Diagrams 
* Behavioral Diagrams
  * Use case diagram
  * Sequence Diagram
